# DACUT 개인정보처리방침

시행일: 2026-02-05  
문의: angelyrlove40@gmail.com

DACUT(이하 “앱”)은 사용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수합니다. 본 개인정보처리방침은 앱 사용 과정에서 어떤 정보가 수집·이용·저장·공유되는지 안내합니다.

## 1. 수집하는 정보
앱은 다음 정보를 처리할 수 있습니다.

1) 사용자가 생성/저장한 콘텐츠  
- 스크린샷 이미지  
- OCR로 추출된 텍스트  
- 노트(제목, 본문, 태그, 상태 등)

2) 기능 수행을 위해 필요한 입력 정보  
- 클립보드 텍스트(사용자가 해당 기능을 활성화한 경우)

3) 선택적 외부 연동 시 전송되는 정보  
- Notion 연동: 사용자가 선택한 노트 내용  
- Google Cloud Vision: OCR 처리를 위해 이미지(사용자가 API 키를 등록한 경우)

## 2. 정보 이용 목적
- 스크린샷/텍스트를 노트로 저장 및 관리  
- 검색/필터/정렬 등 노트 기능 제공  
- 사용자가 선택한 외부 연동 기능 제공

## 3. 보관 및 저장
- 기본적으로 모든 데이터는 사용자 기기에만 저장됩니다.  
- 외부 전송은 사용자가 직접 선택한 기능(예: Notion, Cloud OCR)을 사용할 때만 발생합니다.  

## 4. 제3자 제공 및 처리위탁
앱은 원칙적으로 사용자의 정보를 제3자에게 제공하지 않습니다.  
다만, 사용자가 선택한 외부 서비스 연동 시 해당 서비스로 정보가 전송될 수 있습니다.

- Notion: 사용자가 선택한 노트 내용 전송  
- Google Cloud Vision: OCR 처리를 위해 이미지 전송  

## 5. 권한 사용에 대한 안내
앱은 다음 권한을 기능 제공을 위해 사용할 수 있습니다.

- 접근성 서비스: 클립보드 변경 감지(텍스트 캡처 기능)  
  - 다른 앱의 화면 내용/입력 내용을 수집하지 않습니다.
- 다른 앱 위에 표시(오버레이): 플로팅 메뉴 표시  
- 미디어 프로젝션/캡처: 사용자가 직접 캡처 버튼을 눌렀을 때만 동작  
- 알림: 캡처 상태 및 처리 결과 안내

## 6. 데이터 삭제
- 앱 내에서 개별 노트를 삭제할 수 있습니다.  
- 앱 삭제 시 기기 내 저장된 데이터는 함께 삭제됩니다.

## 7. 아동의 개인정보
앱은 아동을 대상으로 하지 않으며, 고의로 아동의 개인정보를 수집하지 않습니다.

## 8. 방침 변경
법령 또는 서비스 변경에 따라 본 방침은 변경될 수 있으며, 변경 시 앱 또는 스토어 페이지를 통해 안내합니다.
